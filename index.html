Useful HTML snippets



Code Why we use it?
<div> Defines a division or section in an HTML document. It is often used as a container for other HTML elements.
<p> Defines a paragraph. It is used to group sentences and paragraphs of text in a structured way.
<h1> to <h6> Defines HTML headings, with <h1> being the highest level (most important) and <h6> the lowest.
<a> Defines a hyperlink, which is used to link from one page to another.
<ul> & <li> <ul> defines an unordered list, and <li> defines a list item. Together, they create bullet-point lists.
<form> Defines an HTML form for user input. It contains form elements like text fields, checkboxes, and buttons.
<input> Defines an input field within a form where users can enter data. It supports various types like text, password, and submit.
<img> Embeds an image in an HTML page. The 'src' attribute is used to specify the image file location.
<table> Defines a table in HTML, used to organize data into rows and columns.
<span> Defines a section of text with no particular meaning, often used for styling or grouping inline elements.


Useful CSS snippets



Code Why we use it?
color Sets the text color of an element. Example: color: #333;
background-color Sets the background color of an element. Example: background-color: #f0f0f0;
font-size Sets the size of the text. Example: font-size: 16px;
margin Sets the space outside the border of an element. Can be defined for all sides or individually. Example: margin: 20px;
padding Sets the space inside the border of an element. Can be defined for all sides or individually. Example: padding: 10px;
border Sets the border around an element. Includes width, style, and color. Example: border: 1px solid #ccc;
display Specifies how an element is displayed on the page, such as block, inline, or flex. Example: display: flex;
position Specifies the positioning method of an element. Values include static, relative, absolute, and fixed. Example: position: absolute;
width Sets the width of an element. Can be in pixels, percentages, or other units. Example: width: 100%;
height Sets the height of an element. Can be in pixels, percentages, or other units. Example: height: 200px;


Command Line



Code Why we use it?
ls Lists files and directories in the current directory. Example: ls -la (lists all files with detailed info)
pwd Displays the full path of the current working directory. Example: pwd (might output something like /home/username/projects)
cd Changes the current directory. Example: cd /path/to/directory
mkdir Creates a new directory. Example: mkdir new_folder
rm Removes files or directories. Example: rm file.txt (to remove a file), rm -r folder_name (to remove a directory)
cp Copies files or directories. Example: cp source.txt destination.txt
mv Moves or renames files or directories. Example: mv old_name.txt new_name.txt
touch Creates an empty file or updates the timestamp of an existing file. Example: touch newfile.txt
cat Displays the content of a file. Example: cat file.txt
grep Searches for a specific pattern in files. Example: grep 'search_term' file.txt
chmod Changes the file permissions. Example: chmod 755 script.sh
ps Displays currently running processes. Example: ps aux (shows detailed process info)


DITA



Code Why we use it?
<topic> The primary element in DITA, used to define a single unit of content. It can contain other elements like <title>, <body>, and <section>.
<title> Defines the title of a DITA topic or section. It is typically the first element within a <topic>.
<body> Contains the main content of a DITA topic. It includes other elements like <p>, <section>, and <table>.
<p> Defines a paragraph of text within a DITA topic. It is used for adding narrative content.
<section> Divides a DITA topic into logical sections. Each section can contain a <title> and a <body>.
<fig> Used to include a figure, such as an image, within a DITA topic. It typically contains a <title> and <image>.
<table> Defines a table within a DITA topic. It contains elements like <tgroup>, <row>, <entry> to structure tabular data.
<ul> & <li> Defines an unordered list (<ul>) and its items (<li>) within a DITA topic. Used for bullet-point lists.
<xref> Defines a cross-reference link to another topic or section within DITA content. It is used for linking related information.
<codeblock> Displays a block of code within a DITA topic. It preserves formatting and is used for technical documentation.
